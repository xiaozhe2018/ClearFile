Privacy Policy

Last updated: April 2026

Summary (TL;DR)

  • ClearFile runs 100% locally. No data ever leaves your Mac.
  • We do not collect, store, transmit, or sell any personal data.
  • No analytics. No telemetry. No tracking. No ads.
  • The only network request is optional license key verification (Gumroad API).

Data We Do NOT Collect

  • File names, paths, or contents on your disk
  • Browser history, cookies, or browsing activity
  • Personal identifiers (name, email, IP address)
  • Device identifiers or hardware information
  • Usage statistics or crash reports
  • Location data

License Verification

When you activate a Pro license key, ClearFile sends the key to Gumroad's API (api.gumroad.com) to verify its validity. This is the only network request the app makes. The request contains:

  • Your license key
  • The product ID

No other data is transmitted. If you use an offline license key (format CF-XXXX-XXXX-XXXX), no network request is made at all.

Local Storage

ClearFile stores the following data locally on your Mac (in ~/Library/Application Support/ClearFile/ and UserDefaults):

  • License key (if activated)
  • Scheduled clean rules
  • Protection whitelist paths
  • Clean history records
  • Trial start date

All data stays on your Mac. Uninstalling the app removes everything.

File Access

ClearFile requires access to your file system to scan and clean files. With Full Disk Access granted, it can scan protected directories like ~/Library/Mail. Without it, scanning is limited to user-accessible directories.

All file deletions go to macOS Trash (never permanent deletion). You can undo within 30 seconds or recover from Trash manually.

Third-Party Services

  • Gumroad (license verification only) — Gumroad Privacy Policy

No other third-party SDKs, analytics, or services are integrated.

GDPR & International

Since ClearFile does not collect any personal data, GDPR, CCPA, and similar privacy regulations are satisfied by design. There is no data to access, export, or delete because we never collect it.

Children's Privacy

ClearFile does not collect data from anyone, including children under 13.

Changes

We may update this policy. Changes will be posted on this page with an updated date.
