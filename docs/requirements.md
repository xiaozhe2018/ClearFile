# 产品需求文档 — Mac 全盘扫描清理桌面软件

> **产品定位**：帮助 Mac 用户一键扫描磁盘、安全清理垃圾文件、释放磁盘空间的原生桌面应用。

---

## 用户故事

### US-01 磁盘空间概览
作为普通 Mac 用户，我需要一眼看清磁盘哪些地方占用了空间，以便快速决定清理哪些内容。
- Given 应用已打开 / When 点击"开始扫描" / Then 30 秒内展示分类磁盘占用图（系统/应用/文档/缓存/其他）

### US-02 系统垃圾清理
作为 Mac 用户，我需要安全删除系统缓存、日志和临时文件，以便不损坏系统地释放空间。
- Given 扫描完成 / When 选择"系统垃圾"并点击清理 / Then 删除 ~/Library/Caches, /tmp, 系统日志，显示释放空间大小

### US-03 大文件扫描
作为 Mac 用户，我需要找到磁盘上最大的文件和文件夹，以便手动决策删除高价值大文件。
- Given 扫描完成 / When 进入"大文件"模块 / Then 列出 >100MB 的文件，支持排序、右键在 Finder 中显示、一键删除

### US-04 重复文件查找
作为 Mac 用户，我需要找出并删除重复的照片、文档、视频，以便不保留冗余副本。
- Given 指定扫描目录 / When 扫描完成 / Then 以哈希比对呈现重复文件组，保留一个、删除其余

### US-05 应用完整卸载
作为 Mac 用户，我需要卸载 App 并同时清除其在 ~/Library 等处的残留文件，以便彻底释放空间。
- Given 应用列表已加载 / When 选中 App 点击卸载 / Then 移除 .app 及所有关联 plist/cache/support 文件，显示清理大小

### US-06 开发者工具清理（P0 for dev users）
作为开发者，我需要清理 Xcode DerivedData、Docker 镜像、node_modules 等开发缓存，以便快速回收数十 GB 空间。
- Given 检测到开发工具 / When 进入"开发者清理"模块 / Then 列出各工具缓存大小，可选择性清理

### US-07 清理结果报告
作为 Mac 用户，我需要看到每次清理的汇总报告，以便了解清理前后的对比和历史记录。
- Given 清理完成 / When 清理结束 / Then 展示"本次清理：XXX MB，磁盘剩余：XXX GB"并可导出

## MVP 边界

**包含（P0，目标 5 周内完成）：**
- 磁盘空间概览（饼图/树状图可视化）
- 系统垃圾清理（系统缓存、日志、Trash、下载文件夹旧文件）
- 大文件扫描（>50MB，支持筛选、Finder 定位、删除）
- 重复文件查找（哈希比对，支持照片/文档/视频）
- 应用完整卸载（扫描残留 + 一键清除）
- 开发者清理（Xcode DerivedData、npm/yarn cache、Docker builder cache）
- 清理汇总报告（每次清理记录）

**不包含（P1/P2，推迟原因）：**
- P1：浏览器缓存（浏览器私有路径多，兼容性复杂）
- P1：语言包清理（系统完整性保护 SIP 限制，需 root）
- P1：定时清理（增加后台常驻逻辑，复杂度 x2）
- P2：菜单栏常驻（需要独立进程管理）
- P2：隐私痕迹清理（法律合规风险较高）

**单人 MVP 目标周期：≤ 5 周**

## 模块拆解

### 前台模块（macOS App UI）

| 模块 | 原子需求 | 优先级 |
|------|---------|--------|
| 主界面 / 概览 | 磁盘使用饼图、各类别占用 bar、"开始扫描"按钮、上次扫描时间 | P0 |
| 扫描引擎 | 后台并发扫描、实时进度条、可取消 | P0 |
| 系统垃圾 | 分类列表（缓存/日志/Trash/下载）、预览、可勾选清理 | P0 |
| 大文件 | 文件列表排序、类型过滤、Finder 定位、批量删除 | P0 |
| 重复文件 | 相似文件组展示、预览对比、保留策略选择 | P0 |
| 应用卸载 | 已安装 App 列表、残留扫描结果、卸载确认 | P0 |
| 开发者工具 | 检测已安装开发工具、缓存大小展示、分项清理 | P0 |
| 清理报告 | 历史清理记录列表、空间趋势图 | P0 |

### 系统权限层

| 权限 | 用途 | 获取方式 |
|------|------|---------|
| 完整磁盘访问（Full Disk Access） | 扫描 ~/Library, /private/var | 引导用户去系统偏好设置手动授权 |
| 辅助功能（Accessibility，可选） | 获取 App 完整名称 | 仅卸载模块需要 |
| 文件读写 | 删除缓存/临时文件 | 无需额外权限（用户空间目录） |

### 后台服务层

| 模块 | 职责 |
|------|------|
| ScanEngine | 并发递归扫描，输出 FileNode 树 |
| CleanEngine | 按规则删除文件，记录操作日志 |
| DuplicateFinder | SHA-256 哈希比对，分组去重 |
| AppUninstaller | 扫描 App bundle ID 关联路径 |
| DevToolsScanner | 识别 Xcode/Docker/npm 等已知路径 |
| ReportStore | SQLite 持久化清理历史 |

## 合规与风险

| 风险 | 说明 | 应对 |
|------|------|------|
| 误删系统文件 | 清理范围严格限定用户空间，系统保护区不碰 | 白名单路径配置，SIP 保护区只读不写 |
| App Store 审核 | Apple 限制第三方应用访问 /System 等目录 | 优先 Notarize + Direct Distribution，规避 Sandbox 限制 |
| 清理误删用户数据 | 重要文件被误删无法恢复 | 清理前展示预览列表，删除前二次确认，不直接 rm（先移入 Trash） |
| macOS 版本兼容 | API 差异 | 最低支持 macOS 13 Ventura（SwiftUI 稳定版） |
| 用户隐私 | 文件路径不上传云端 | 全本地运行，无联网数据收集 |
